# Whisker Coding Challenge 
Task: Find the shortest distance between any of the points listest in the csv file and prints it's associated sn numbers

## Getting the answer from project
1. Open terminal or command prompt in computer
2. Install latest version of [Node](https://nodejs.org/en/download) or if already installed, use a package version manager like [NVM]() or [N] to make sure you version number is 20 or higher.
3. In the root of the project (whisker-test/), run `npm install`.
4. Then run `node longitude-lat.j`.
5. Answer should show in the terminal
